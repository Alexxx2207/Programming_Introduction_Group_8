- казаха ли ви какво означават настройките в командата g++ -std=c++17 -Wall -Wextra -Wpedantic <cpp-file>
- за какво сте говорили?
- отказах се да искам да предавате задачи до края на упражнението - когато - тогава, колкото - толкова.
- реших да ви групирам задачите не в 3, а в две групи: 
    1. едната група е задачи по материала за седмицата. Очаквам да ги решите в рамките на една седмица. 
    2. другата група е бонус за напреднали/знаещи, като си свършат първата секция(ако въобще сте я пипнали). Там може да се иска все още невзет материал. Не гарантирам, че всяка седмица ще я има - зависи колко съм зает.
- задачите и в двете секции не са наредени по сложност - важно умение на контролно е да избирате кои задачи да решите първо, т.е. да прецените кои са ви лесни, защото понякога няма да ви стига времето.

## Какво ще изведе кода

```cpp
#include<iostream>

int main() {
    int x = 10;
    if(x += x -= 10)
        std::cout << (x *= 20);
    else
        std::cout << x;

	return 0;
}
```


```cpp
#include<iostream>

int main() {
    int x = 10;

    std::cout << ((x == 10) || (x / 0));

	return 0;
}
```

```cpp
#include<iostream>

int main() {
    int x = 10;

    std::cout << ((x == 20) || (x / 0));

	return 0;
}
```

```cpp
#include <iostream>
int main() {
    int i = 1;

    std::cout << (i + ++i) << "\n";
}
```

```cpp
#include <iostream>

int main() {
    int i = 1;

    std::cout << (i + i++) << "\n";
}
```

```cpp
#include <iostream>
int main() {
    char c = -1;
    
    unsigned int u = c;

    std::cout << (u > 0) << "\n";
}
```


## Извън преподадения материал / на когото му е скучно

### Задача 1

**_За тази задача НЕ може да ползвате допълнителни библиотеки, примерно: `<cstring>` `<string>` и `<regex>`. Напишете си всичко сами .Трябва да ползвате `char*` - тоест динамично заделяне и освобождаване на памет с `new` и `delete`._**

Получавате от стандартния вход първо дължина на низ `str`, после низа `str` и накрая низ-шаблон `pattern` с максимална дължина 1000 ASCII символа, като `@` и `#` имат специално значение:
- `#` означава точно един произволен символ
- `@` означава нула или повече произволни символа

Проверете дали `str` пасва на `pattern`.

Пояснение: `str` **не** съдържа `@` и `#`.

Примери:
```
Input: str = "xx", pattern = "x"
Output: false
```

```
Input: str = "gfjka", pattern = "gf##a"
Output: true
```

```
Input: str = "abfa", pattern = "@a"
Output: true
```
